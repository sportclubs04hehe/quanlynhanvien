Đổi Mật Khẩu

@if (hasError('currentPassword', 'required')) {
Mật khẩu hiện tại là bắt buộc
}
@if (hasError('newPassword', 'required')) {
Mật khẩu mới là bắt buộc
} @if (hasError('newPassword', 'minlength')) {
Mật khẩu phải có ít nhất 6 ký tự
}
@if (hasError('confirmPassword', 'required')) {
Xác nhận mật khẩu là bắt buộc
} @if (hasError('confirmPassword', 'passwordMismatch')) {
Mật khẩu xác nhận không khớp
}
Lưu ý: Sau khi đổi mật khẩu thành công, bạn sẽ được đăng xuất và cần đăng nhập lại bằng mật khẩu mới.
